Information about apis

Defining endpoint-:
1./upload-:this is a post api used for uploading image in cloudinary
2./info-:this is post api used for uploading lost car-info in database
3./car/status/:car_number-: this is get request for getting car info by car_number
4./police/signup-:this is post api for police to create a account with their info
5./police/login-:this is post api for police to login with valid credentials
6./police/info/:id-:this is get request for fetching police info with id
7./car/detail/:id-:this is get request for fetching car info with id
8./car/found-:this api is used to update the car data if found
